## Descrição
Este é o backend em Nest.JS do Projeto Estuda.AI, uma aplicação que visa fornecer um board kanban para que estudantes possam organizar seus estudos para diversas provas

## Instalação

```bash
$ npm install
```

## Executar Projeto

```bash
$ npm run build
$ npm run dev
```
